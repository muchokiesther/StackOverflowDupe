How to remove the box around the link element?

I'm new to HTML and CSS. I would like to have an invisible link, but I don't want to set the style of 'a' tags directly. Instead, I would like to set the style of its parent element so that it becomes invisible.

Answers:

The 'box' around your link has a default outline property defined. Be sure to include outline: none; to any element or pseudo-selector that includes this treatment.
